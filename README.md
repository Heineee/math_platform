## ✨ 功能特性
- **核心计算**  
  - 线性代数：矩阵运算、行列式计算、特征值求解
  - 微积分：符号微分、数值积分、极限计算
- **算法库**  
  - 数值方法：牛顿迭代法、龙格-库塔法、蒙特卡洛模拟
  - 优化算法：梯度下降、遗传算法
- **可视化**  
  - 2D/3D图形绘制（集成Matplotlib）
  - 动态交互式图表（支持Jupyter Notebook）
- **扩展接口**  
  - 插件式架构，支持自定义算法模块
  - 兼容NumPy数组格式
